# LifeLink - Emergency Healthcare Locator

LifeLink is a real-time emergency healthcare platform that helps users find nearby hospitals, ambulances, blood donors, and emergency services.

## Features

* Nearby Hospital Search
* Ambulance Tracking
* Blood Donor Finder
* SOS Emergency Alerts
* Secure User Authentication
* Responsive Design

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS

### Backend

* Node.js

### Database



## Installation

```bash
git clone <repository-url>
cd life-link
npm install
npm run dev
```
